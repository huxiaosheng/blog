# VIM编辑器

i:切换到输入模式

:wq 保存并推出

![vi-vim-cheat-sheet-sch](/Users/huyuan/Documents/GitHub/Blog/images/vi-vim-cheat-sheet-sch.gif)





### 创建文件或多个文件

```shell
vim aa bb cc
```



### 打开文件

```
vi +3 aa //打开文件光标跳转到第3行
```



### 底行模式常用指令


| 命令        | 功能        |
| --------- | --------- |
| **:w**    | 保存        |
| **:q**    | 退出        |
| **:!**    | 强制        |
| **:ls**   | 列出所有打开的文件 |
| **:n**    | 跳转到下一个文件  |
| **:15**   | 光标跳转到15行  |
| **:/xxx** | 向后查找      |
| **:?xxx** | 向前查找      |



### 命令模式常用指令

| 命令         | 功能                    |
| ---------- | --------------------- |
| **h**      | 光标左移                  |
| **j**      | 光标下移                  |
| **k**      | 光标上移                  |
| **i**      | 光标右移                  |
| **ctrl+f** | 向下翻页（front）           |
| **ctrl+b** | 向上翻页（back）            |
| **ctrl+d** | 向下翻半页（down）           |
| **ctrl+u** | 向上翻半页（）               |
| **dd**     | 删除光标所在行               |
| **o**      | 在光标所在行的下方插入一行并切换到输入模式 |
| **yy**     | 复制光标所在行               |
| **p**      | 在光标所在行的下方粘贴           |
| **P**      | 在光标所在行的上方粘贴           |
